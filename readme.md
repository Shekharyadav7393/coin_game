# Coin Collector Game
This is a simple 2D browser game created using JavaScript and the Phaser 3 framework.
The purpose of this project was to practice basic game development concepts such as
player movement, collision detection, timers, and simple enemy behavior.
## Features
- Player movement using arrow keys
- Coin collection with score tracking
- Countdown timer
- A basic enemy that follows the player
- Start and Game Over screens

## Tech Used
- JavaScript
- Phaser 3
## Controls
Use arrow keys to move the player.
## Goal
Collect as many coins as possible before time runs out and avoid the enemy.
